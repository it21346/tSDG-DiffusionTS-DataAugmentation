Οδηγίες Εκτέλεσης:

- Πηγαίνετε στο directory it2022057
- Κατεβάζεται τα nodeJS modules εκτελώντας 'npm i', καθώς έχω προσθέσει στα 
dependencies του package.json τις απαραίτητες πληροφορίες
- Τρέχετε το τοπικό server side πατώντας 'node app.js' ή 'npm start'
- Ανατρέχετε σε έναν browser της επιλογής σας και πληκτρολογείτε το url 
'http://localhost:3000'
- Το πρόγραμμα είναι έτοιμο και μπορείτε να το ελέγξετε
